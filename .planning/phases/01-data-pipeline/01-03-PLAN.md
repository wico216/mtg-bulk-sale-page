<task type="auto">
  <name>Task 1: Create build-time data generation script</name>
  <files>
    scripts/generate-data.ts
    package.json
  </files>
  <action>
    **scripts/generate-data.ts:**
    Create the main build script that orchestrates the full pipeline.

    1. Import `parseAllCsvFiles` from `../src/lib/csv-parser`
    2. Import `enrichCards` from `../src/lib/enrichment`
    3. Import `writeFileSync`, `mkdirSync`, `existsSync` from `node:fs`
    4. Import `CardData` from `../src/lib/types`
    5. Import `resolve` from `node:path`

    Main function:
    a. Resolve paths relative to project root (use `process.cwd()`)
    b. Parse all CSVs: `const rawCards = parseAllCsvFiles(resolve('data/inventory'))`
    c. Log: `Found ${rawCards.length} cards from CSV files`
    d. Enrich: `const { cards, stats } = await enrichCards(rawCards)`
    e. Ensure `data/generated/` directory exists
    f. Build the CardData object:
       ```
       {
         cards: cards,
         meta: {
           lastUpdated: new Date().toISOString(),
           totalCards: cards.length,
           totalSkipped: stats.skipped,
           totalMissingPrices: stats.missingPrices
         }
       }
       ```
    g. Write to `data/generated/cards.json` with `JSON.stringify(data, null, 2)`
    h. Print summary (per user decision, exact format): `Processed ${stats.processed} cards, ${stats.skipped} skipped (no match), ${stats.missingPrices} missing prices`
    i. Print: `Output: data/generated/cards.json (${cards.length} cards)`

    Wrap main() in try/catch that logs errors and exits with code 1.
    Call `main()` at the end.

    **package.json:**
    Add npm scripts:
    - `"generate"`: `"tsx scripts/generate-data.ts"` -- run pipeline standalone
    - Update `"build"`: `"tsx scripts/generate-data.ts && next build"` -- chain pipeline before Next.js build

    This ensures the data pipeline runs automatically before every build, and can also be run independently during development.
  </action>
  <verify>
    Run `npm run generate` and verify:
    1. Console shows "Found X cards from CSV files"
    2. Console shows progress updates during enrichment
    3. Console shows summary line: "Processed X cards, Y skipped (no match), Z missing prices"
    4. `data/generated/cards.json` exists and contains valid JSON with cards array and meta object
    5. `cat data/generated/cards.json | npx tsx -e "const d = require('./data/generated/cards.json'); console.log('Cards:', d.cards.length, 'Updated:', d.meta.lastUpdated);"` shows correct data
  </verify>
  <done>generate-data.ts runs the full pipeline (CSV parse -> Scryfall enrich -> JSON output). npm run generate produces data/generated/cards.json with enriched cards and metadata. Build summary prints to console.</done>
