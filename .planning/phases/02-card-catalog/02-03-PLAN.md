  - On mobile: stacked layout (image on top, text below)
  - Mobile full screen: `max-md:fixed max-md:inset-0 max-md:rounded-none max-md:max-w-none max-md:overflow-y-auto`

  - Container with `aspect-[5/7]` and `relative` for next/image fill
  - On desktop: `md:w-64 md:flex-shrink-0` for fixed width
  - On mobile: `w-full max-w-xs mx-auto` for centered image
  - If imageUrl null: same placeholder as tile but larger
- Text section:
  - Card name: `text-xl font-semibold mb-2`
  - Set name + rarity: `text-sm text-zinc-500 mb-3` -- format: "{setName} -- {rarity}" (capitalize rarity)
  - Foil status: if foil, show "Foil" badge same style as tile but slightly larger
  - Price: `text-lg font-medium mb-3` -- "$X.XX" or "N/A"
  - Condition + quantity: use same display name mapping as card-tile (NM, LP, etc.), format: "{condition} -- Qty: {quantity}"

- useEffect for scroll lock: when component mounts, set `document.body.style.overflow = 'hidden'`. On unmount (cleanup), set it back to `''`. This prevents background scrolling while modal is open.
  - Since this uses useEffect, add 'use client' directive to this component OR handle the scroll lock in card-grid.tsx instead. Decision: handle scroll lock in card-grid.tsx since it already has 'use client'.
