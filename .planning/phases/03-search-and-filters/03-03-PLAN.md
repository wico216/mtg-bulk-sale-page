     - Overflow: `overflow-y-auto`
   - Sheet header: drag handle indicator (small gray bar centered at top: `w-10 h-1 bg-zinc-300 dark:bg-zinc-600 rounded-full mx-auto mt-3 mb-4`)
   - Sheet content with `px-5 pb-6 space-y-5`:
     - Section "Colors": `<p className="text-xs font-medium text-zinc-500 uppercase tracking-wide mb-2">Colors</p>` then `<ManaColorPills />`
     - Section "Set": label then `<MultiSelect label="Set" options={uniqueSets} selected={selectedSets} onToggle={toggleSet} />`
     - Section "Rarity": label then `<MultiSelect label="Rarity" options={uniqueRarities} selected={selectedRarities} onToggle={toggleRarity} formatOption={formatRarity} />`
     - Section "Sort": label then `<SortDropdown />`
     - If hasActiveFilters(): `<button onClick={() => { clearFilters(); onClose(); }} className="w-full py-2 text-sm text-accent font-medium">Clear all filters</button>`

7. Scroll lock: use useEffect keyed on `isOpen`:
   ```
   useEffect(() => {
     if (isOpen) {
       document.body.style.overflow = 'hidden';
     } else {
       document.body.style.overflow = '';
     }
     return () => { document.body.style.overflow = ''; };
   }, [isOpen]);
   ```
   Note: this works alongside the existing modal scroll lock in card-grid.tsx because both set the same property. When the bottom sheet closes, the modal lock may still be in effect if a modal is open (unlikely since bottom sheet is for mobile browsing). The cleanup function handles unmount.
