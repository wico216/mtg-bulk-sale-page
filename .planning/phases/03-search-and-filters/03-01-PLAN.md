  <action>
Run `npm install zustand` to add Zustand 5 to the project.

Create `src/lib/store/` directory and `src/lib/store/filter-store.ts`:

1. Import `create` from `zustand` and `Card` type from `@/lib/types`
2. Export a `SortOption` type: `'price-desc' | 'price-asc' | 'name-asc'`
3. Define `FilterState` interface with:
   - `allCards: Card[]` -- source data set once on mount
   - `searchQuery: string` -- text search input
   - `selectedColors: Set<string>` -- mana color filter (W, U, B, R, G, C)
   - `selectedSets: Set<string>` -- set/expansion filter
   - `selectedRarities: Set<string>` -- rarity filter
   - `sortBy: SortOption` -- current sort order
   - Actions: `setAllCards`, `setSearchQuery`, `toggleColor`, `toggleSet`, `toggleRarity`, `setSortBy`, `clearFilters`
   - Derived: `getFilteredCards(): Card[]`, `hasActiveFilters(): boolean`

4. Create store with `create<FilterState>()((set, get) => ({...}))` (curried pattern for TypeScript in Zustand 5):
   - Default `sortBy`: `'price-desc'` (price high-to-low per user decision)
   - All Sets initialize as `new Set<string>()`
   - `setAllCards`: simple setter
   - `setSearchQuery`: simple setter
   - `toggleColor`, `toggleSet`, `toggleRarity`: each creates a NEW Set instance (critical for Zustand reactivity -- mutating existing Set does not trigger re-render). Pattern:
     ```
     toggleColor: (color) => set((state) => {
       const next = new Set(state.selectedColors);
       next.has(color) ? next.delete(color) : next.add(color);
       return { selectedColors: next };
     }),
     ```
   - `setSortBy`: simple setter
   - `clearFilters`: resets searchQuery to `''`, all Sets to `new Set<string>()`, sortBy to `'price-desc'`

5. Implement `getFilteredCards()` using `get()` to access current state:
   - Start with `let result = get().allCards`
   - **Name search**: if `searchQuery.trim()` is non-empty, filter by `card.name.toLowerCase().includes(query.toLowerCase())`
   - **Color filter**: if `selectedColors.size > 0`:
     - Check if `selectedColors.has('C')` (wants colorless)
     - Collect non-C color codes: `[...selectedColors].filter(c => c !== 'C')`
     - Card passes if: (wantsColorless AND card.colorIdentity.length === 0) OR (colorCodes.length > 0 AND card.colorIdentity.some(ci => colorCodes.includes(ci)))
     - This is OR logic per user decision
   - **Set filter**: if `selectedSets.size > 0`, filter by `selectedSets.has(card.setName)`
   - **Rarity filter**: if `selectedRarities.size > 0`, filter by `selectedRarities.has(card.rarity)`
   - **Sort**: spread into new array then sort:
     - `'price-desc'`: null prices to end, then `b.price! - a.price!`
     - `'price-asc'`: null prices to end, then `a.price! - b.price!`
     - `'name-asc'`: `a.name.localeCompare(b.name)`
   - Return sorted result

6. Implement `hasActiveFilters()`: returns true if searchQuery is non-empty OR any Set has size > 0 OR sortBy !== 'price-desc'

Export `useFilterStore` and `SortOption` type.
