## Critical Anti-Patterns (do NOT repeat these)

- **Worktree absolute-path drift** → When spawning a gsd-executor in a worktree, the agent's `cd <main-repo-path>` in any Bash invocation poisons subsequent cwd-dependent calls. Mitigation: rely on `git -C <worktree>` plus relative paths from `git rev-parse --show-toplevel` inside the worktree, OR use sequential mode for single-plan waves.
- **Single-mock concurrency blind spot** → A mock that returns a single boolean per process cannot exercise cross-request locking semantics. The advisory-lock no-op shipped past tests because nothing in the test suite called the lock from two different driver sessions.
