1. Extract the `loadCardData()` function from `src/app/page.tsx` into a shared utility:
   ```
   import { readFileSync } from "node:fs";
   import { resolve } from "node:path";
   import type { CardData } from "@/lib/types";

   export function loadCardData(): CardData | null {
     try {
       const filePath = resolve(process.cwd(), "data/generated/cards.json");
       const raw = readFileSync(filePath, "utf-8");
       return JSON.parse(raw) as CardData;
     } catch {
       return null;
     }
   }
   ```

4. Handle card not found (stale cart item): if `card` is undefined, render a "No longer available" row:
   - Gray background, card name replaced with `cardId`, a remove button
   - `<div className="flex items-center gap-3 p-3 bg-zinc-50 dark:bg-zinc-900 rounded-lg">...`

5. Normal cart item layout -- a horizontal row:
   - Thumbnail: `<Image src={card.imageUrl} alt={card.name} width={48} height={67} className="rounded object-cover flex-shrink-0" />` (48x67 is ~5:7 aspect ratio matching card dimensions). If no imageUrl, show a small gray placeholder div.
   - Middle section (flex-1 min-w-0):
     - Card name: `<p className="text-sm font-medium truncate">{card.name}</p>`
     - Set name: `<p className="text-xs text-zinc-400 truncate">{card.setName}</p>`
     - Unit price: `<p className="text-sm text-zinc-600 dark:text-zinc-300">{card.price !== null ? `$${card.price.toFixed(2)}` : 'N/A'}</p>`
   - Quantity controls (right side, flex-shrink-0):
     - +/- stepper buttons AND an editable number input (per user decision: cart page has BOTH)
     - Layout: `<div className="flex items-center gap-1.5">`
     - Minus button: `<button onClick={() => quantity <= 1 ? onRemove() : onQuantityChange(quantity - 1)}>` with `-` icon. Same circular button style as tile stepper.
     - Number input: `<input type="number" value={quantity} min={1} max={maxStock} className="w-12 text-center text-sm border border-zinc-300 dark:border-zinc-600 rounded-md py-1 bg-transparent [appearance:textfield] [&::-webkit-outer-spin-button]:appearance-none [&::-webkit-inner-spin-button]:appearance-none" onChange={handleInputChange} />`
     - Plus button: `<button onClick={() => onQuantityChange(quantity + 1)} disabled={quantity >= maxStock}>` with `+` icon
     - Stock warning: `{stockWarning && <p className="text-xs text-amber-600 mt-0.5">Only {maxStock} available</p>}`
   - Remove button (far right): `<button onClick={onRemove} className="p-1.5 text-zinc-400 hover:text-red-500 transition-colors" aria-label="Remove from cart">` with an X/trash icon

6. `handleInputChange` function:
   - Parse input value with `parseInt(e.target.value, 10)`
   - If `isNaN(parsed)` or `parsed <= 0`: call `onRemove()` (per decision: minus below 1 removes)
   - If `parsed > maxStock`: call `onQuantityChange(maxStock)`, show stock warning briefly (set `stockWarning` to true, setTimeout to clear after 2 seconds)
   - Otherwise: call `onQuantityChange(parsed)`

3. Read cart state:
   - `const items = useCartStore((s) => s.items)` -- the full Map (needed to iterate)
   - `const setQuantity = useCartStore((s) => s.setQuantity)`
   - `const removeItem = useCartStore((s) => s.removeItem)`
   - `const clearCart = useCartStore((s) => s.clearCart)`
   - `const totalItems = useCartStore((s) => s.totalItems())`

4. Handle hydration flash: use `useCartStore.persist.hasHydrated()` to prevent showing "Your cart is empty" before localStorage has loaded. Create a state:
   ```
   const [hydrated, setHydrated] = useState(false);
   useEffect(() => {
     const unsub = useCartStore.persist.onFinishHydration(() => setHydrated(true));
     if (useCartStore.persist.hasHydrated()) setHydrated(true);
     return unsub;
   }, []);
   ```
   If `!hydrated`, show a minimal loading state (e.g., a few skeleton lines or just an empty div -- keep it simple).

5. Build a card lookup Map for O(1) access: `const cardMap = useMemo(() => new Map(cards.map(c => [c.id, c])), [cards])`

6. Convert cart items Map to an array for rendering: `const cartEntries = useMemo(() => [...items.entries()], [items])`

7. Compute `totalPrice`: `const totalPrice = useMemo(() => { let sum = 0; for (const [id, qty] of items) { const card = cardMap.get(id); if (card?.price) sum += card.price * qty; } return sum; }, [items, cardMap])`

8. Empty cart state (when hydrated and items.size === 0):
   ```
   <div className="max-w-3xl mx-auto px-4 text-center py-16">
     <p className="text-lg text-zinc-500 mb-4">Your cart is empty</p>
     <Link href="/" className="inline-block px-5 py-2 text-sm font-medium rounded-md bg-accent text-white hover:bg-accent-hover transition-colors">
       Browse cards
     </Link>
   </div>
   ```

9. Cart items list (when items.size > 0):
   - Page title: `<h1 className="text-xl font-semibold mb-6">Your Cart</h1>`
   - Clear cart button (top right): `<button onClick={handleClearCart} className="text-sm text-zinc-400 hover:text-red-500 transition-colors">Clear cart</button>`
   - `handleClearCart`: calls `window.confirm('Are you sure you want to clear your cart?')`. If confirmed, calls `clearCart()`. Using native confirm per research ("Don't Hand-Roll" section -- simple, accessible, no state to manage).
   - Items container: `<div className="space-y-3">` then map over `cartEntries`:
     ```
     cartEntries.map(([cardId, qty]) => (
       <CartItem
         key={cardId}
         cardId={cardId}
         quantity={qty}
         card={cardMap.get(cardId)}
         maxStock={cardMap.get(cardId)?.quantity ?? 0}
         onQuantityChange={(newQty) => setQuantity(cardId, newQty, cardMap.get(cardId)?.quantity)}
         onRemove={() => removeItem(cardId)}
       />
     ))
     ```
   - Render `<CartSummaryBar totalItems={totalItems} totalPrice={totalPrice} />` at the bottom (it's fixed-position so renders anywhere)
