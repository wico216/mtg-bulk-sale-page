  <name>Task 1: Create cart store with Zustand persist middleware and Map-based state</name>
  <files>src/lib/store/cart-store.ts</files>
  <action>
Create `src/lib/store/cart-store.ts`:

1. Import `create` from `zustand` and `persist`, `createJSONStorage` from `zustand/middleware`

2. Define `CartState` interface:
   - `items: Map<string, number>` -- card ID to quantity mapping
   - Actions: `addItem(cardId: string, maxStock?: number)`, `removeItem(cardId: string)`, `setQuantity(cardId: string, qty: number, maxStock?: number)`, `clearCart()`
   - Derived methods: `totalItems(): number`, `hasItem(cardId: string): boolean`, `getQuantity(cardId: string): number`

3. Create store with curried pattern (same as filter-store.ts): `create<CartState>()(persist((set, get) => ({...}), options))`

4. Implement actions -- CRITICAL: always create a new Map instance for reactivity (same pattern as Set reactivity in filter-store.ts):
   - `addItem(cardId, maxStock?)`: create new Map from state.items, compute `current + 1`, cap at `maxStock` if provided, set new Map
   - `removeItem(cardId)`: create new Map, delete cardId, set new Map
   - `setQuantity(cardId, qty, maxStock?)`: create new Map, if `qty <= 0` delete the entry, else set `Math.min(qty, maxStock ?? qty)`, set new Map
   - `clearCart()`: set `{ items: new Map() }`

5. Implement derived methods using `get()`:
   - `totalItems()`: sum all values in `get().items`
   - `hasItem(cardId)`: `get().items.has(cardId)`
   - `getQuantity(cardId)`: `get().items.get(cardId) ?? 0`

6. Configure persist middleware options:
   - `name: "viki-cart"` (localStorage key)
   - `storage: createJSONStorage(() => localStorage, { replacer, reviver })` -- this is SSG-safe (createJSONStorage wraps in try/catch, returns undefined when localStorage unavailable)
   - `replacer`: if value is Map, convert to `{ __type: "Map", entries: [...value] }`, else return value as-is
   - `reviver`: if value has `__type === "Map"`, reconstruct `new Map(value.entries)`, else return value as-is
   - `partialize: (state) => ({ items: state.items })` -- only persist the items Map, not actions or derived methods

Export `useCartStore`.
  </action>
